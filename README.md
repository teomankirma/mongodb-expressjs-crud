# MongoDB Express CRUD

## Setup

1. Install dependencies:

   ```bash
   npm install
   ```

2. Create a `.env` file based on `.env.example` and update the values:

   ```bash
   cp .env.example .env
   ```

3. Start the application:

   ```bash
   npm run dev
   ```
